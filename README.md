# Employee Payroll & Attendance Tracker

## Overview
An Excel-based payroll and attendance management system built using Microsoft Excel.

## Features
- Employee payroll calculations
- Attendance tracking
- Automatic salary calculation
- Net salary calculation
- Interactive dashboard
- PivotTables
- Pivot Charts
- Department-wise employee analysis

## Tools Used
- Microsoft Excel
- Formulas
- PivotTables
- Pivot Charts
- Dashboard Design

## Skills Demonstrated
- Excel
- Data Analysis
- Dashboard Creation
- Payroll Management
- Data Visualization
